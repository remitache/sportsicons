Advantages

One file, over 100 Icons

In a single collection, Sports Icons offers you hunderds of icons.

No JavaScript Required

Fewer compatibility concerns because Sports Icons doesn't require JavaScript.

Perfect on Retina Displays

Sports icons are vectors, which mean they're gorgeous on high-resolution displays.

Up to date icons

No update required. Teams and other icons are updated so that you don't have to handle anything.

Infinite Scalability

Scalable vector graphics means every icon looks awesome at any size.

CSS Control

Easily style icon size, shadow, and anything that's possible with CSS.

Get Started

  1. Paste the following code into the <head> section of your site's HTML.

    <link rel="stylesheet" href="http://remitache.github.io/sportsicons/css/sportsicons.min.css">

  2. Check out the examples bellow to include the icons in your webpage

  3. Copy and paste this exemple to display an icon

    <span class="si si-championsleague"></span>

  4. You can add option to size it up (24, 36, 48, 96, 128, 256 or you can custom the font-size by yourself):

    <span class="si si-championsleague si-96"></span>

    Or invert the color for black icon over black background

    <span class="si si-rugby-newzealand si-96 si-invert"></span>
